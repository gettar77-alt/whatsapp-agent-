# تركيب جسر مها (الجهاز المرتبط)

يربط مها كجهاز مرتبط برقم صاحب النشاط. مها ترد على **الأرقام غير المحفوظة فقط**،
وتتجاهل المحفوظين بالاسم. نفس الرقم، نفس المحادثات، صفر فقدان بيانات.

> ⚠️ يحتاج صاحب الرقم حاضر بجواله (لمسح QR لمرة واحدة).
> تطبيق واتساب بزنس لازم يكون محدّث لآخر إصدار.

نفّذ على السيرفر (نفس الشاشة السوداء):

## ١) تثبيت Node.js + مكتبات تشغيل المتصفح
```bash
apt update && apt install -y nodejs npm
apt install -y ca-certificates fonts-liberation libasound2t64 libatk-bridge2.0-0 \
  libatk1.0-0 libcairo2 libcups2 libdbus-1-3 libdrm2 libexpat1 libfontconfig1 \
  libgbm1 libglib2.0-0 libgtk-3-0 libnspr4 libnss3 libpango-1.0-0 libx11-6 \
  libxcomposite1 libxdamage1 libxext6 libxfixes3 libxkbcommon0 libxrandr2 libxshmfence1
```

## ٢) سحب الكود وتثبيت المكتبات
```bash
cd /opt/maha && git pull
cd /opt/maha/whatsapp-bridge && npm install
```
(npm install يحمّل المتصفح المدمج — ياخذ دقيقة - دقيقتين)

## ٣) تشغيل أول مرة لمسح QR (صاحب الرقم حاضر)
```bash
cd /opt/maha/whatsapp-bridge && node bot.js
```
- يطلع كود QR في الشاشة
- صاحب الرقم يفتح: **واتساب بزنس ← الإعدادات ← الأجهزة المرتبطة ← ربط جهاز** ويمسحه
- لما يطلع **"مها متصلة وجاهزة"** → نجح الربط
- اضغط `Ctrl+C` لإيقافه مؤقتاً

## ٤) تشغيله كخدمة دائمة (٢٤ ساعة)
```bash
cat > /etc/systemd/system/maha-bridge.service <<'EOF'
[Unit]
Description=Maha WhatsApp Linked-Device Bridge
After=network.target maha.service

[Service]
WorkingDirectory=/opt/maha/whatsapp-bridge
ExecStart=/usr/bin/node bot.js
Restart=always
User=root

[Install]
WantedBy=multi-user.target
EOF
systemctl daemon-reload
systemctl enable --now maha-bridge
systemctl status maha-bridge --no-pager
```

## التحكم (تشغيل/إيقاف مها)
```bash
systemctl stop maha-bridge      # إيقاف مها (صاحب الرقم يرد بنفسه)
systemctl start maha-bridge     # تشغيل مها
journalctl -u maha-bridge -f    # متابعة السجل لحظياً
```
