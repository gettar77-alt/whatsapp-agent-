# مجلد صور الشقق

كل شقة لها مجلد فرعي. مها ترسل كل الصور اللي داخل المجلد لما العميل يطلب صور تلك الشقة.

| الشقة | المجلد | علامة مها |
|---|---|---|
| بينت هاوس دورين (الأكبر) | `media/penthouse/` | `[[photos:penthouse]]` |

## كيف تضيف صور لشقة
1. ارفع الصور (jpg / png) داخل مجلد الشقة، مثلاً `media/penthouse/`
2. على السيرفر: `cd /opt/maha && git pull && systemctl restart maha-bridge`

> نصيحة: اختر أفضل ٦ إلى ٨ صور لكل شقة — أحسن من إرسال عدد كبير دفعة وحدة.
> أسماء الصور ترتّب أبجدياً، فلو تبي ترتيب معيّن سمّها `01.jpg`, `02.jpg`...
